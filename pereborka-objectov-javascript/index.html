Как узнать количество пар "ключ": "значение" в Объекте JavaScript

Самый простой способ подсчитать, это перебрать все свойства объекта с помощью for in и создать отдельную переменную которая послужит счетчиком, и она будет подсчитывать сколько свойтсв было внутри объекта.

На примере предидущего кода добавим переменную counter 

                    
    let data = {
    name: 'Jhon',
    age: 34,
    colors: {
        fontColor: 'red',
        backGround: 'black',
            }
        };
    let counter = 0;
    for (let key in data) {
    if (typeof(data[key]) === 'object') {
    for (let i in data[key]) {
        console.log(`Свойство ${i} имеет значение ${data[key][i]}`);
        counter++;
            }
        } else {
        console.log(`Свойство ${key} имеет значение ${data[key]}`);
        counter++;
            }
        }
    console.log(counter);

Ответ будет равен 4

Есть еще один метод, кторый называется Object.keys(data) Данный метод возвращает массив данных с количеством ключей. Тут важно помнить что и у объектов и у массивов данных есть свойство length которое выводит точное количество элементов. 

                    
    Object.keys(data).length

Небольшой итог.

Объекты это структуры которые могут сохранять в себе абсолютно любые типы данных в формате "ключ": "значение" Могут быть так же вложенные объекты, такие как объекты в объекты и массивы в объекты и т.д. Чтобы их перебрать мы можем использовать конструкцию for in Так же получать свойства можно как через точку так и через квадратные скобки. Еще у объектов есть встроенные методы и встроенные свойства которые мы можем очень удобно использовать при работе. Ну и самое главное, чтобы наш объект умел что то делать, мы можем записывать фунции прямо внутри объекта. и таким образом создаются методы объекта.
