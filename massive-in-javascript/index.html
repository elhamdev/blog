Массивы в JavaScript

Создано - 25 июня 2022

Создание и структура данных в массиве

Массивы данных это структуры которые содержат элементы по порядку. Это может быть массив с разными типами данных. Все свойства которые содержат массивы можно вывести с помощью команды 

            
console.dir

Для эксперементов создадим массив. Массивы создаются через квадратные скобки. 

            
let array = [1, 2, 3, 4, 6, 8];

Нумерация данных в массиве данных начинается с 0. Стоит помнить об этом!

Часто при работе с массивами данных, нужно либо добавить либо удалить свойства. Либо в начале массива либо в конце. Для начала возьмем методы которые работают именно с концом массива. Обращемся к перемменной в которой лежит этот массив. 

            
array.pop();

Первый метод называется 'pop' на самом деле он всего лижь удаляет последний массив из нашего элемента. 

            
console.log(array);

При выполнении кода увидим что наш массив стал меньше на один элемент.

Для добавления элемента нужно использовать следующий метод, и следует задать в аргументах внутри скобок, что мы хотим добавить. 

            
array.push(9);

Я задал число 9 которое будет пушиться в конец массива.

Если же говорить про начало массива то существует такие методы как 

            
array.shift();
array.unshift(15);

Метод array.shift(); удаляет первое свойство в массиве

Метот array.unshift(15); добавляет в начало свойств массива данные из из скобок которые указать в виде аргументов.

В реальной практике shift() и unshift() очень редко используются. Помимо добавления элементов в начало и конец массива, часто требуется перебрать данные внутри массива. Для переборки можно использовать цикл for() 

            
let array = [ 1, 2, 3, 4, 5];
    
for (let i = 0; i < array.length; i++) {
    console.log(array[i]);
}

Так же для переборки можно использовать цикл for of 

            
let array = [1, 2, 3, 4, 5];
    
for (let value of array) {
    console.log(value);
}

Данный метод ничем не отличается от цикла for()

Конструкция for of() работает только с массиво-подобными сущностями и именно поэтому мы не можем перебирать им объекты.

Теперь немного про свойсво length() 

            
let array = [1, 2, 3, 4, 6, ];
console.log(array.length);

Ответ получим 5.

Метод forEach()

А теперь речь пойдет про самую часто используемую конструкцию этот метод называется forEach() Метод forEach() делает практически тоже самое что и for() и for of() но с одним существенным отличием. forEach() можно более тонко настроить.

Метод forEach() принимает внутри себя callback функцию. 

            
let array = [1, 2, 3, 4, 6, ];
    
array.forEach(function(item, number, array) {
    console.log(`${number}: ${item} внутри массива ${array}`);
});

Данный метод может принимать внутри себя три аргумента. Первый аргумент - это item, это тот самый элемент который перебираем. И назвать данный аргумент можно как угодно. Следующий аргумент - это номер по порядку внутри массива. Третий аргумент - это ссылка на тот массив который мы перебираем. Получим такой ответ: 

            
0: 1 внутри массива 1,2,3,4,5
1: 2 внутри массива 1,2,3,4,5
2: 3 внутри массива 1,2,3,4,5
3: 4 внутри массива 1,2,3,4,5
4: 5 внутри массива 1,2,3,4,5

Данный метод позволяет удобно манипулировать любым массивом данных.

Метод split

Метод split() позволяет создавать массив из строк, например: 

            
let str      = prompt('', '');
let prods    = str.split(', ');
    
console.log(prods);

В ответ получим массив из любых данных введеных пользователем.

Метод join()

Для того чтобы разделить массив, нужно использовать метод join() явно указав разделитель. 

            
let str      = prompt('', '');
let prods    = str.split(', ');
    
console.log(prods.join('; '));

В этом примере метод join() превратил массив в строку.

Метод sort()

Метод сортировки предназначен для сортировки данных внутри массива 

            
let str      = prompt('', '');
let prods    = str.split(', ');
prods.sort();
console.log(prods);

Все строковые данные введенные через prompt() отсортируются в алфавитном порядке. Следует помнить что метод sort() всегда сортирует элементы как строки. И когда следует отсортировать данные состоящие из чисел нужно добавлять функцию в аргумент метода. 

            
let str      = prompt('', '');
let prods    = str.split(', ');
prods.sort(sortNums);
console.log(prods);
    
function sortNums(a, b){
    return a - b; 
}

Применив данную функцию метод sort() будет правильно работать и с числами.

Псевдомассивы

По факту псевдомассивы - это такие же массивы как и обычные за одним исключением, у псевдомассивов нет никаких методов. Это просто структура которая хранит данные просто по пордку.
