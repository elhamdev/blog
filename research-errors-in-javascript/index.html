Поиск ошибок в JavaScript

Создано - 17 июля 2022

Консоль разработчика, это инструмент с которым разработчики сталкиваются постоянно.

Вызывается он с помощью "Инспектора Кода".

Либо просто нужно нажать правой кнопкой мыши и выбрать пункт "Исследовать Код Страницы".

В консоли есть множество вкладок, первые три. Для примера используем данный кусок кода: 

                
function hello() {
    console.log('Hello World');
}
hello();
                
function hi () {
    console.log('Say Hi');
}
hi();
                
let arr = [1, 4, 14, 40, 45],
    sorted = arr.sort(compareNum);

function compareNum(a, b) {
    return a - b;
}

Первая вкладка это - Elements В ней находится всё содержимое HTML тегов, CSS элементов, hover и before элементов, а так же классы.

Дальше идёт вкладка Sources В ней находятся все папки и все файлы в проекте, которые нам доступны.

Открыв наш код во вкадке Sources, можно проставить брейкпоинты и посмотреть каждый шаг выполнения программы.

С помощью дебаггера можно посмотреть каждый шаг сравнивания внутри функции compareNum из примера нашего кода.

Так же брейкпоинты можно проставить вручную внутри кода. Для этого следует прописать ключевое слово 

            
function hello() {
    console('Hello World');
    debugger; // брейкпоинт проствленный вручную
}
hello();

Теперь код будет останавливаться на месте где написано debugger;

Вот таким образом можно отлавливать свои ошибки в консоли разработчика.
