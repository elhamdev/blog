Динамическая типизация в JavaScript

Создано - 18 июля 2022

Типы данных в JavaScript

Простые типы данных: Числа, строки, boolean, null, undefined, symbol.

Объекты: Массивы, функции, объект даты, регулярные выражения, ошибки.

Теперь когда познакомились со всеми типами данных, можно поговорить про "Динамическую типизацию" Это возможность одного типа данных превращаться в другой.

Т.е. число может стать строкой, строка числом.

Также стоит помнить что в других языках программирования есть и статическая типизация данных, когда число всегда остаётся числом.

Первый способ, чтобы превратить любой тип данных в строку, можно использовать "To String" например 

            
typeof(String(null));  // null превратится в строку
typeof(String(4));     // число превратится в строку

Второй способ называется конкотинация, это когда мы какой то тип данных складываем со строкой: 

            
typeof(5 + '');        // снова тип данных станет строкой

Не большой код для примера 

            
let num = 5;
console.log('http://vk.com/feed/' + num);
let fontSize = 26 + 'px';

Теперь немного о том как превратить любой тип данных в число

Первый способ похож на String только нужно использовать Number 

            
typeof(number('Hi'));  // типом данных будет число

Второй способ это "унарный плюс". Говоря простыми словами это когда перед какими то данными ставить знак + 

            
typeof(+'5');   //  тут тоже типом данных станет число

Например если мы получаем данные от пользователей через prompt и нужно преобразовать в числовой тип данных 

            
let answer = +prompt('Hello', ''); // Получим number в любом случае

Третий способ преобразования "To Boolean" true либо false. Нравда или ложь.

Типы данных, которые всегда будут выдавать false: 

            
0, 'string', null, undefined, NaN; // всегда false

Пример кода как переключателя на сайте 

            
let switcher = null;
if (switcher) {
    console.log('Working..');  // Не сработает
}
switcher = 1;
if (switcher) {
    console.log('Working..');  // Сработает 
}

Важно помнить, что всё что мы получаем от пользователей на сайте - всё это строки! И на основании этих знаний нужно понимать, что иногда нужно будет преобразовывать полученную информацию в другой тип данных.
