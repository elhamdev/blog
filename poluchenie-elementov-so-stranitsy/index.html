Получение элементов со страницы в JavaScript

Создано - 19 июля 2022

Первое с чем необходимо познакомиться, это такое понятие как - DOM. Или 'Document Object Module' - Объектная модель документа.

Весь HTML код является обычным документом. Этот документ представлен виде дерева узлов. При этом узлы связаны между собой отношениями "Родительский - дочерний"

Например у тега body - есть множество дочерних элементов, а у каждого дочернего элемента есть только один родитель - это body.

И такая структура вложенности создаётся во время вёрстки. Когда такой документ сформирован, мы его можем представить в виде объекта.

Именно поэтому и называется - Объектная модель документа. И как и у любого объекта у него есть свои методы.

Именно эти методы мы и будем использовать, для того чтобы иметь доступ к определённым элементам на страничке.

Чтобы обратиться к DOM нужно использовать такую сущность как 'document'

Самый распространённый метод называется getElementById

На нашей HTML страничке мы можем назначать уникальные идентификаторы 'id' и обращаться к ним. 

            
let box = document.getElementById('box');
console.log(box); // выведет соержимое переменной

По правилам id может быть только один на всей странице.

Помимо этого можно получить доступ к элементам по названию тегов.

Но одинаковых тегов на странице может быть огромное количество.

При обращении по тегам получим эти элементы в виде коллекции - псевдомассиве.

Важно помнить, что у псевдомассивов нет методов. 

            
let btns = document.getElementsByTagName('button');
console.log(btns);

Для точного обращения используя данный метод, следет писать так: 

            
let btns = document.getElementsByTagName('button')[1];
console.log(btns);

Помимо тегов, так же можно использовать и классы. 

            
let btns = document.getElementsByClassName('btn');

Самый современный и часто используемый метод называется 'querySelectorAll' Во внутрь он принимает CSS селекторы 

            
let btns = document.querySelectorAll('.btn');
console.log(btns);

В отличие от остальных способов, querySelectorAll имеет метод работы который назывется forEach();

При использовании данного метода не стоит забывать ставить точку! 

            
btns.forEach(item => {
    console.log(item);
});

Есть похожий метод который называется просто 'querySelector' но он позволяет получить доступ только к первому элементу.
