Объекты

Объекты в JavaScript это так называемые ассоциативные массивы, которые содержатся в других языках программирования, например в php.

Создадим для примера простой объект. Объект в JavaScript создается через фигурные скобки. 

                    
    const obj = {
    name: 'test',
    width: 1024,
    height: 1024,
    colors : {
        border: 'black',
        background: 'red',
        }
    };

Обект готов, если нужно получить информацию про какую либо из свойств из значений объекта, нужно использовать точечный синтаксис, либо синтаксис при помощи квадратных скобок. 

                    
    console.log(obj.name);
    console.log(obj['name']);

В ответ вернется информация из строки name = test.

Для удаления какого либо значения из объекта используется оператор delete. Например 

                    
    delete obj.name;
    console.log(obj);

Таким образом просто удалить значение из объекта. Чтобы перебрать все возможные данные внутри объекта нужно использовать конструкцию for in. 

                    
    for (let key in obj) {
        console.log(`Свойство ${key} имеет значение ${obj[key]}`);
    }

Получим вот такой ответ: 

                    
    Свойство name имеет значение test
    Свойство width имеет значение 1024
    Свойство height имеет значение 1024
    Свойство colors имеет значение [object Object]

Для получения данных внутри вложенного объекта нужно запустить перебор for in внутри перебора с помощью условной конструкции if. 

                    
    for (let key in obj) {
    if (typeof(obj[key]) === 'object') {
    for (let i in obj[key]) {
        console.log(`Свойство ${i} имеет значение ${obj[key][i]}`);
        }
    }else {
        console.log(`Свойство ${key} имеет значение ${obj[key]}`);
        }
    }

Ответ будет таким: 

                    
    Свойство name имеет значение test
    Свойство width имеет значение 1024
    Свойство height имеет значение 1024
    Свойство border имеет значение black
    Свойство background имеет значение red
