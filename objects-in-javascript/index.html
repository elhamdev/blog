Объекты в JavaScript

Создано - 15 июня 2022

Объекты в JavaScript это так называемые ассоциативные массивы, которые так же содержатся в других языках программирования, например в php.

Создание и переборка объектов

Создадим для примера простой объект. Объект в JavaScript создается через фигурные скобки. 

                    
    const obj = {
    name: 'test',
    width: 1024,
    height: 1024,
    colors : {
        border: 'black',
        background: 'red',
        }
    };

Обект готов, если нужно получить информацию про какую либо из свойств из значений объекта, нужно использовать точечный синтаксис, либо синтаксис при помощи квадратных скобок. 

                    
    console.log(obj.name);
    console.log(obj['name']);

В ответ вернется информация из строки name = test.

Для удаления какого либо значения из объекта используется оператор delete. Например 

                    
    delete obj.name;
    console.log(obj);

Таким образом просто удалить значение из объекта. Чтобы перебрать все возможные данные внутри объекта нужно использовать конструкцию for in. 

                    
    for (let key in obj) {
        console.log(`Свойство ${key} имеет значение ${obj[key]}`);
    }

Получим вот такой ответ: 

                    
    Свойство name имеет значение test
    Свойство width имеет значение 1024
    Свойство height имеет значение 1024
    Свойство colors имеет значение [object Object]

Для получения данных внутри вложенного объекта нужно запустить перебор for in внутри перебора с помощью условной конструкции if. 

                    
    for (let key in obj) {
    if (typeof(obj[key]) === 'object') {
    for (let i in obj[key]) {
        console.log(`Свойство ${i} имеет значение ${obj[key][i]}`);
        }
    }else {
        console.log(`Свойство ${key} имеет значение ${obj[key]}`);
        }
    }

Ответ будет таким: 

                    
    Свойство name имеет значение test
    Свойство width имеет значение 1024
    Свойство height имеет значение 1024
    Свойство border имеет значение black
    Свойство background имеет значение red

Метобы и свойства объектов

Как узнать количество пар "ключ": "значение" в Объекте JavaScript

Самый простой способ подсчитать, это перебрать все свойства объекта с помощью for in и создать отдельную переменную которая послужит счетчиком, и она будет подсчитывать сколько свойтсв было внутри объекта. 

                    
    let data = {
    name: 'Jhon',
    age: 34,
    colors: {
        fontColor: 'red',
        backGround: 'black',
            }
        };
    let counter = 0;
    for (let key in data) {
    if (typeof(data[key]) === 'object') {
    for (let i in data[key]) {
        console.log(`Свойство ${i} имеет значение ${data[key][i]}`);
        counter++;
            }
        } else {
        console.log(`Свойство ${key} имеет значение ${data[key]}`);
        counter++;
            }
        }
    console.log(counter);

Ответ будет равен 4

Есть еще один метод, кторый называется Object.keys(data) Данный метод возвращает массив данных с количеством ключей. Тут важно помнить что и у объектов и у массивов данных есть свойство length которое выводит точное количество элементов. 

                    
    Object.keys(data).length

Небольшой итог

Объекты это структуры которые могут сохранять в себе абсолютно любые типы данных в формате "ключ": "значение" Могут быть так же вложенные объекты, такие как объекты в объекты и массивы в объекты и т.д. Чтобы их перебрать мы можем использовать конструкцию for in Так же получать свойства можно как через точку так и через квадратные скобки. Еще у объектов есть встроенные методы и встроенные свойства которые мы можем очень удобно использовать при работе. Ну и самое главное, чтобы наш объект умел что то делать, мы можем записывать фунции прямо внутри объекта. и таким образом создаются методы объекта.
